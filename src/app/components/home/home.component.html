Depuis 2021, rentalux vous accompagne dans la location de toutes sortes de véhicules. Quelque soit votre besoin, rentalux est le partenaire de votre réussite. Pour réserver un véhicule, il vous suffit de créer un compte (gratuit) et de vous connecter. Vous pourrez ensuite choisir le véhicule adéquat et le réserver à une date libre. Le véhicule sera livré sur le drop site de votre choix afin de vous offrir un maximum de liberté.
